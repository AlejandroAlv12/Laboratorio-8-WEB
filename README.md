# Laboratorio-8-WEB

>¿Cuál es la diferencia entre autenticación y autorizacion?

autenticación verifica quien es el usuario como por ejemplo por el ´ID´. La autorización determina que puede hacer ese usuario autenticado  es decir define sus permisos de acceso a recursos o funciones. La autenticación siempre ocurre primero.

>¿Cuál es la función del token JWT en la guía?

Su función es servir como una prueba de identidad y permisos para la autorización. 
Permite que el usuario autenticado acceda de forma segura a recursos y servicios limitados, ya que el token contiene la información de quién es el usuario y qué tiene permitido hacer, sin necesidad de volver a verificar las credenciales en cada solicitud
